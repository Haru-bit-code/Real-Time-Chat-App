# 💬 Real-Time Chat App

Live multi-user chat built with Python, Flask, and WebSockets.
No account needed — just enter a name and start chatting.

## Live Demo
🌐 [Try it here](your-render-link)

## Features
- Real-time messaging with WebSockets
- Multiple users simultaneously
- No login required
- Shows who joined the chat
- Last 20 messages load for new users

## Tech Stack
- Python · Flask · Flask-SocketIO · WebSockets · HTML/CSS/JS

## Run Locally
pip install -r requirements.txt
python app.py
